# NexMart 管理端 | NexMart Admin Frontend

本仓库为 [NexMart 电商平台](https://github.com/你的用户名/NexMart) 的管理端前端，基于 Vue 3 + Element Plus 构建。

This is the admin-facing frontend of the [NexMart E-Commerce Platform](https://github.com/你的用户名/NexMart), built with Vue 3 + Element Plus.

## 快速启动 | Quick Start

```bash
npm install
npm run dev
```

访问 http://localhost:8086
